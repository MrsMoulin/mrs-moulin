## Etiquette Class

The two hour etiquette class is a one of the kind workshop, offering boys and girls an opportunity to learn table manners in any dining setting. During the class they will be guided through a formal table setting, experience a four-course meal while using each utensil properly. These courtsies will help them become more confident at any table. Ages 6+

## Cooking Class

My hands on cooking classes are personally curated to the individual. Each class teaches proper food handling, knife skills, and proper cooking techniques. Most classes are 3-course meal driven, which includes an appetizer, main dish and dessert in a three hour class. Ages 10+
