## Etiquette Class

The two hour etiquette class is a one of the kind workshop, offering boys and girls an opportunity to learn table manners in any dining setting. During the class they will be guided through a formal table setting, enjoy a four-course meal while using each utensil properly. These courtsies will help the become more confident at any table. Ages 6+

My cooking classes are personally curated to the individual. Each class teaches proper food handling, knife skills, and proper cooking techniques. Most classes are 3-course meal driven, including an appetizer, main meal and dessert in a three hour class. Ages 10+
